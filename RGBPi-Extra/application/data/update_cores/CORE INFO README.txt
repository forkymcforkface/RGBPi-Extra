Mesen replaces fceumm in this release, fceumm handles sound emulation for FDS poorly and 
is generally less accurate. This specific version of Mesen incorporates the Audio Balancing 
changes detailed here: https://github.com/riggles1/MesenAudioBalance

You can place .so, .7z, .zip files in here and they will show up in the core updates portion of the ui.
